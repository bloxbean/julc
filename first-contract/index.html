Write Your First JuLC Contract

This tutorial walks you through writing your first Cardano smart contract with JuLC. You will install the CLI, scaffold a project, write a spending validator, compile it to UPLC, and test it locally — all in under 10 minutes.

  • julc CLI installed (see below)

CLI Command Reference

Section titled “CLI Command Reference”
CommandDescription
julc new <name>Create a new project (-t basic, -t gradle, -t maven)
julc buildCompile validators to UPLC + generate CIP-57 blueprint
julc checkDiscover and run on-chain tests
julc replInteractive REPL with live budgets
julc eval <expr>Evaluate a single expression
julc blueprint inspect <file>Inspect a compiled blueprint
julc blueprint address <file>Compute script address from blueprint
julc uplc decode <hex>Decode UPLC from hex/CBOR
julc uplc fmt <file>Pretty-print UPLC
julc installInstall/update stdlib sources
julc --versionShow version
