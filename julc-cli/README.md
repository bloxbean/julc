# julc-cli - JuLC CLI

Command-line tool for building Cardano smart contracts in Java.

## Usage

```bash
# Show available commands
julc --help

# Create a new project
julc new <project-name>

# Build validators (compile Java to UPLC)
julc build

# Show version
julc --version
```
